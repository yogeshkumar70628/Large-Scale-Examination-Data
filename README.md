# Exploratory Data Analysis on Large-Scale Examination Data

## Overview

This project presents an end-to-end Exploratory Data Analysis (EDA) of a large-scale examination dataset containing over **2.33 million records** collected across India.

The analysis focuses on uncovering:
- score distribution patterns
- regional performance variation
- center-level benchmarking
- statistical outlier detection
- data quality insights
- feature-engineered performance segmentation

The project demonstrates practical large-scale data analysis using Python and industry-standard analytical libraries.

# Key Insights

- Strong regional variation exists in average examination performance.
- Certain examination centers consistently outperform others.
- Score segmentation reveals concentration in both lower and higher performance bands.
- The dataset contains minimal missing values and no exact duplicate records.
- Extreme score ranges warrant further domain-level validation.
